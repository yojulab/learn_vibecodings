## 프로젝트 목적

이 프로젝트는 다양한 AI CLI, MCP 서버, 그리고 Vibe Coding IDE 환경을 설치하고, 사용법을 정리하며, 실제 프로젝트 구현 예시를 제공하는 것을 목표로 합니다.

## 주요 목표
- AI 관련 커맨드라인 툴(CLI) 설치 및 활용법 정리
- Model Context Protocol(MCP) 서버 구축 및 실습
- Vibe Coding IDE 환경 설정 및 사용법 안내
- 실습 중심의 프로젝트 예제 구현
